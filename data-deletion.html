Data Deletion – Account Book App

Account Book provides users with full control over their data. Users can delete data at multiple levels directly from within the app.

1. Delete Individual Transactions

Users can delete individual financial transactions without deleting their user account.

  1. Open the Account Book app
  2. Select an account
  3. Long Press a transaction
  4. Tap Delete icon and confirm

2. Delete an Account (Inside the App)

Users can create multiple accounts (for example: Personal or Business). Any account can be deleted independently.

  • The selected account is permanently removed
  • All transactions under that account are permanently deleted
  • The user login remains active

3. Delete User Account (Login Deletion)

Users can permanently delete their entire Account Book user account from the app User Profile.

  • User authentication data (Firebase Authentication)
  • All accounts created by the user
  • All associated transactions and data
Important: All deletions are permanent and irreversible.
